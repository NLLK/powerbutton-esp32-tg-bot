# Powerbutton ESP32 Telegram Bot

Простой Telegram-бот для удалённого включения ПК, работающий на ESP32 и библиотеке FastBot2 от AlexGyver. Он замыкает контакты кнопки питания на разъёме передней панели материнской платы с помощью оптопары. Проект ставит перед собой следующие цели:

- Открытый исходный код с либеральной лицензией
- Понятные схемы подключения и инструкции для любителей
- Удобный и настраиваемый интерфейс пользователя
- Поддержка нескольких языков (легко расширяемая)
- Безопасность через белый список разрешённых пользователей
- Задокументированые функции, с которыми будет работать пользователь

# Программное обеспечение

## Конфигурация

Основные настройки задаются в файле `include/env.h`. В целях безопасности этот файл **не** включён в репозиторий (он содержит данные Wi-Fi и токен бота). Вы должны создать собственный `env.h` из шаблона `include/env.h.default`:

1. Скопируйте `include/env.h.default` и переименуйте его в `include/env.h`.
2. Заполните параметры в соответствии с таблицей:

| Параметр        | Где взять                                                                                                         | Пример                                      |
| --------------- | ----------------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| `WIFI_SSID`     | Имя вашей Wi-Fi сети (скрытые сети тоже работают).                                                                | `MyHomeWiFi`                                |
| `WIFI_PASS`     | Пароль от Wi-Fi (может быть пустым, если сеть открыта).                                                           | `secret123`                                 |
| `BOT_TOKEN`     | Получите токен у [@BotFather](https://t.me/BotFather) при создании бота.                                          | `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11` |
| `ADMIN_CHAT_ID` | Ваш личный ID чата (используется для важных уведомлений). Узнайте его у [@userinfobot](https://t.me/userinfobot). | `123456789`                                 |

После создания `env.h` с правильными значениями скомпилируйте и прошейте устройство. Затем отправьте `/start` вашему боту (тому, имя которого вы выбрали у BotFather). Если бот отвечает (иногда это занимает несколько секунд), значит всё настроено верно.

## Команды

Бот понимает следующие команды. Для удобства они также доступны в виде кнопок в reply-клавиатуре. Пользователи без доступа могут использовать только `/start`, `/help` и `/request_access`. В меню бота (слева от поля ввода) всегда есть ссылки на основные команды.

| Команда                 | Описание                                                                                                                                                   |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `/start`                | Показать главное меню с кнопками.                                                                                                                          |
| `/help`                 | Показать справку.                                                                                                                                          |
| `/turn_on`              | Имитировать короткое нажатие кнопки питания – включить/выключить ПК.                                                                                       |
| `/long_press`           | Имитировать длительное нажатие кнопки питания – принудительно выключить ПК.                                                                                |
| `/test_ping`            | Проверить, отвечает ли бот, и измерить время отклика.                                                                                                      |
| `/get_config`           | Показать текущие настройки бота.                                                                                                                           |
| `/set_settings`         | Открыть меню настроек.                                                                                                                                     |
| `/set_lang`             | Сменить язык. Выберите язык в меню или введите один из: `EN`, `RU`.                                                                                        |
| `/set_short_press_time` | Установить длительность (в мс) короткого нажатия. Бот покажет допустимый диапазон и будет ждать ваш числовой ввод.                                         |
| `/set_long_press_time`  | Установить длительность (в мс) длительного нажатия. Бот покажет допустимый диапазон и будет ждать ваш числовой ввод.                                       |
| `/msg_adm_pwron`        | Включить/отключить уведомление администратора, когда другой пользователь нажимает кнопку питания. Выберите `true` или `false` в меню или введите значение. |
| `/manage_users`         | Открыть меню управления пользователями.                                                                                                                    |
| `/request_access`       | Запросить доступ к боту (для пользователей, у которых его ещё нет).                                                                                        |
| `/clear_wait_list`      | Очистить список ожидающих подтверждения пользователей.                                                                                                     |
| `/revoke_access`        | Отозвать доступ у пользователя. Выберите код пользователя в меню или введите его вручную.                                                                  |
| `/give_access`          | Предоставить доступ пользователю из списка ожидания. Выберите код пользователя в меню или введите его вручную.                                             |
| `BACK`                  | Вернуться в главное меню (то же, что и `/start`).                                                                                                          |

## Продвинутая конфигурация

Некоторые параметры можно изменить только в коде, в файле `include/const/config.h`:

| Параметр                                         | Описание                                                                                     |
| ------------------------------------------------ | -------------------------------------------------------------------------------------------- |
| `LONG_PRESS_TIME_DEFAULT`                        | Длительность длительного нажатия по умолчанию (мс).                                          |
| `LONG_PRESS_TIME_MIN`                            | Минимально допустимая длительность длительного нажатия (мс).                                 |
| `LONG_PRESS_TIME_MAX`                            | Максимально допустимая длительность длительного нажатия (мс).                                |
| `SHORT_PRESS_TIME_DEFAULT`                       | Длительность короткого нажатия по умолчанию (мс).                                            |
| `SHORT_PRESS_TIME_MIN`                           | Минимально допустимая длительность короткого нажатия (мс).                                   |
| `SHORT_PRESS_TIME_MAX`                           | Максимально допустимая длительность короткого нажатия (мс).                                  |
| `LANG_DEFAULT`                                   | Язык по умолчанию (определён в `include/const/translate.h`).                                 |
| `MAX_USERS_ALLOWED_TO_CONTROL`                   | Максимальное количество авторизованных пользователей.                                        |
| `MAX_USERS_IN_WAIT_LIST`                         | Максимальное количество пользователей в очереди на доступ.                                   |
| `SEND_ADMIN_MSG_ON_OTHER_USERS_POWER_ON_DEFAULT` | Отправлять ли администратору уведомление, когда другой пользователь нажимает кнопку питания. |

Кроме того, в этом файле содержатся пути к полям свойств, используемые библиотекой `Properties` – это полезно, если вы хотите добавить больше настроек или встроить это приложение в другой проект.

# Аппаратная часть

![Схема](./hardware/scheme.png)

Оптопара, управляемая ESP32, замыкает контакты кнопки питания на разъёме передней панели материнской платы.  
Контакты материнской платы подключаются к оптопаре проводами DuPont (2.54 мм), как и провода физической кнопки питания.  
Таким образом, когда вы нажимаете физическую кнопку, она замыкает контакты, и когда срабатывает оптопара – происходит то же самое.

Я питаю ESP32 через USB-порт на материнской плате, который остаётся под напряжением даже при выключенном ПК.

# Устранение неполадок

## Программная часть

- Убедитесь, что имя Wi-Fi сети, пароль и токен бота указаны верно в `env.h`. После их изменения перепрошейте ESP32.
- Если у вас нет доступа к боту, проверьте правильность ID администратора (это строка из цифр).

## Аппаратная часть

- Если всё собрано по схеме, но не работает, перепроверьте полярность проводов от материнской платы. Они должны быть подключены строго по схеме.
