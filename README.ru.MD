# Powerbutton ESP32 Telegram Bot

Функционально простой Телеграм Бот для включеня ПК, основанный на библиотеке "FastBot2" от AlexGyver. Он работает на ESP32 (я использовал ESP32 C3 Super mini) и коротит межсобой контакты кнопки питания на коннекторе материнской платы с помощью оптопары. Проект не претендует на оригинальность, и предназначен лишь для того, чтобы предоставить сообществу следующие ценности:
- открытый исходный код с доступной для внесения изменений лицензией;
- принципиальные схемы для аппаратной части проекта с описанием для неопытных энтузиастов;
- удобный и простой интерфейс пользователя с гибкой настройкой;
- поддержка мультиязычности с возможностью расширения и добавления новых языков;
- обеспечение безопасности по средством управления списком разрешенных пользователей;
- задокументрированная UX составляющая.

# ПО

## Конфигурация

Основная настройка приложения осуществляется с помощью файла "include/env.h". Его по-умолчанию нет в репозитории, т.к. это то файл, который нужно хранить в секрете. Он содержит в себе пароли и ключи, которые не должны оказаться в публичном доступе, но необходимы для работы и ESP32, и ТГ бота. Создайте свой файл "include/env.h" по шаблону из файла "include/env.h.default". Просто скопируйте его, переименуйте в "env.h", и введите настройки в соответствии с этой таблицей:

| Параметр      | Где его взять                                                                                                                                      | Пример                        |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| WIFI_SSID     | Имя сети Wi-Fi                                                                                                                                     | SomePublicWifiName            |
| WIFI_PASS     | Пароль для этой сети                                                                                                                               | \*\*\*\*\*\*\*\*              |
| BOT_TOKEN     | Воспользуейтесь [этим ботом (BotFather)](https://t.me/BotFather) чтобы создать бота и получить токен                                               | 2143123421:NahIDontGiveYouOne |
| ADMIN_CHAT_ID | ID чата с вами. Нужен чтобы присылать важные уведомления. Можете написать [этому боту (userinfobot)](https://t.me/userinfobot) чтобы узать этот ID | 123123123                     |

После того, как вы создали этот файл и ввели нужные параметры, собрали и прошили устройство, вы можете написать "/start" тому боту, что вы создали (вы вводили имя бота при его создании у BotFather, там будет что-то в стиле @***bot). Если бот отвечает на эту команду (порой, это требует немного времени), значит все было настроено верно. 

Теперь вы можете пользовать ботом. 

## Команды 

Бот знает определенный набор команд, перечисленных и описанных ниже. Для пользователей, у которых нет доступа к приложению, на данный момент доступны только команды "/start", "/help" и "/request_access". Все эти команды доступны в исполнение внезависимости от кнопок, находящихся над клавиатурой в чате, но для удобства в этой клавиатуре всегда прописаны текущие доступные команды. Также в меню бота (кнопка слева от поля ввода) всегда есть ссылки на команды, которые выдают меню и описание команд.

| Команда               | Описание                                                                                                                                                                                             |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| /start                | Показывает меню с кнопками над клавиатурой                                                                                                                                                           |
| /help                 | Отображает справку по приложению                                                                                                                                                                     |
| /turn_on              | Краткое нажатие кнопки питания - для включения и выключения ПК                                                                                                                                       |
| /long_press           | Длительное нажатие кнопки питания - для принудительного выключения ПК                                                                                                                                |
| /test_ping            | Проверка отклика бота - для определения есть ли связь у ESP32, и как долго обрабатывается простейшая команда                                                                                         |
| /get_config           | Получение текущих настроек бота                                                                                                                                                                      |
| /set_settings         | Отобразить меню настроек                                                                                                                                                                             |
| /set_lang             | Установка языка. Выберите язык в меню, или введите одно из следующих: "EN", "RU"                                                                                                                     |
| /set_short_press_time | Установка длительности краткого нажатия. Выведет сообщение с доступным диапазоном значений и предложит в текстовом виде отправить новое время (в мс)                                                 |
| /set_long_press_time  | Установка длительности долгого нажатия. Выведет сообщение с доступным диапазоном значений и предложит в текстовом виде отправить новое время (в мс)                                                  |
| /msg_adm_pwron        | Установка флага, определяющего будет ли оправляться администратору сообщение, если кто-то другой нажал или зажал кнопку. Выберите в меню "true" или "false", или отправьте сообщение с таким текстом |
| /manage_users         | Отобразить меню управления пользователями                                                                                                                                                            |
| /request_access       | Запросить доступ (для пользователей, у которых доступа еще нет)                                                                                                                                      |
| /clear_wait_list      | Очистить список пользователей, ожидающих разрешения на пользование приложением                                                                                                                       |
| /revoke_access        | Отозвать разрешение на пользование приложением. Предложит в меню выбрать код пользователя, но вы можете отправить его сами                                                                           |
| /give_access          | Дать разрешение на пользование приложением. Предложит в меню выбрать код пользователя из списка ожидания, но вы можете отправить его сами                                                            |
| BACK                  | То же что и "/start". Нужно для меню                                                                                                                                                                 |
## Продвинутая конфигурация

Основные настройки можно поменять с помощью команд бота. Но некоторые параметры изменяются только из кода в файле "include/const/config.h". Ниже перечислены все параметры и их значение:

| Параметер                                      | Значение                                                                                                    |
| ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| LONG_PRESS_TIME_DEFAULT                        | Значение по-умолчанию для времени длительного зажатия                                                       |
| LONG_PRESS_TIME_MIN                            | Минимальное значение времени длительного зажатия для настройки через команды                                |
| LONG_PRESS_TIME_MAX                            | Максимальное значение времени длительного зажатия для настройки через команды                               |
| SHORT_PRESS_TIME_DEFAULT                       | Значение по-умолчанию для времени короткого нажатия                                                         |
| SHORT_PRESS_TIME_MIN                           | Минимальное значение времени короткого нажатия для настройки через команды                                  |
| SHORT_PRESS_TIME_MAX                           | Максимальное значение времени короткого нажатия для настройки через команды                                 |
| LANG_DEFAULT                                   | Язык по-умолчанию. Берется из файла "include/const/translate.h"                                             |
| MAX_USERS_ALLOWED_TO_CONTROL                   | Максимальное количество пользователей приложения                                                            |
| MAX_USERS_IN_WAIT_LIST                         | Максимальное количество пользователей, ожидающих разрешения на пользование приложением                      |
| SEND_ADMIN_MSG_ON_OTHER_USERS_POWER_ON_DEFAULT | Флаг, определяющий будет ли оправляться администратору сообщение, если кто-то другой нажал или зажал кнопку |

Так же в этом файле есть настройки для изменения путей к полям для библиотеки "Properties", что может быть полезно в случае если вы планируете добавить настроек, или вообще встроить это приложение в какое-то другое.

# "Железо"

![Scheme](./hardware/scheme.png)

Оптопара, управляемая ESP32, замыкает цепь кнопки питания на коннекторе материнской платы.
Пины на материской плате подключаются с помощью "DuPont" (провода с наконечниками 2.54мм) к оптопаре, как и провода кнопки питания.
Поэтому, когда нажата кнопка питания, она замыкает контакты на материнской плате, как и оптопара.

Я запитываю ESP32 через USB, у которого есть функция работы при выключеном ПК.

# Исправление возможных проблем 
## ПО

- Следует убедиться что имя сети, пароль и токен указаны верно. На всякий случай уточню, что при изменении в коде, нужно и ESP32 перепрошить;
- Если у вас нет доступа к приложению, то проверьте верно ли указан ID пользователя. Это должна быть строка из нескольких чисел.

## "Железо"

- В случае, если всё собрано верно, и схема кажется исправной, проверьте еще раз полярность проводов от материнской платы. Они должны быть подключены строго по схеме.
